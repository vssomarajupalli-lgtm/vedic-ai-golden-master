# ADR-006: Mandali as an Interpretive Overlay (Decoupled from Core Math)

## Context
During Phase 15 preparation, the Mandali layer was identified as a risk if tightly coupled to core deterministic timing analysis. It required architectural decoupling to prevent systemic failures.

## Repository Evidence
"Phase 15 transitions Mandali from a *Mathematical Constraint* (which crashes formulas if missing) into an *Interpretive Overlay* (which enriches the final natural language answer)."

## Decision
The Mandali system shall act strictly as an interpretive overlay, providing nuance rather than acting as a core mathematical constraint for formula evaluation.

## Architectural Rationale
This major architectural pivot ensures that missing or failed Mandali data does not crash core formula timing analysis. It restricts Mandali from mutating the core True/False boolean logic of a formula.

## Consequences
The Answer Composer shall weave Mandali commentary into the deterministic final state as nuance, but the Mandali Engine shall not possess the capability to override the primary mathematical evaluation.

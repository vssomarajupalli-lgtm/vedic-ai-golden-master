# ADR-012: Non-Numeric Confidence Model (Boolean Layering)

## ADR ID

## Status

## Recovery Classification

## Context
During Phase 12 Data Model design, the system needed a framework to evaluate the validity of astrological predictions. A decision was made to avoid arbitrary numeric weights in favor of a more deterministic logical approach.

## Repository Evidence
"- Designed the conceptual Confidence Model, outlining how 5 distinct layers... contribute to the validity of an astrological prediction without utilizing raw numeric scoring yet."

## Decision
Evaluation relies strictly on logical AND/OR gating (Boolean Layering, utilizing concepts like Fatal Denials vs. Positive Alignment) rather than arbitrary numeric weights (e.g., +15% per planet).

## Architectural Rationale
This locks in the core philosophical approach to the mathematical evaluation layer, ensuring predictability and determinism rather than opaque numeric scoring systems.

## Consequences
Mathematical evaluations remain boolean-layered. Developers cannot introduce numeric weights for astrological factors within the evaluation engine.
